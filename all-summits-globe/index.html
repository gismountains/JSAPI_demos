Summits around the globe

The list of the summits is an export from www.geonames.org.
The geonames are catagorized in different topics/classes. For the summits the data are filtered to the class "T mountain,hill,rock,..." with code "MT mountain", "MTS mountains", "PK peak" and "PKS peaks". The attribute altitude is set to "elevation" if existing, otherwise the value is set from "digital elevation model".
